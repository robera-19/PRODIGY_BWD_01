# User CRUD API with Express & Zod

A simple RESTful API to **Create, Read, Update, and Delete users** using **Node.js**, **Express**, and **Zod** for input validation. Users are stored in **in-memory storage** (no database required).

---

## Features

- **Create** a new user with validation
- **Read** all users or a single user by ID
- **Update** user details with validation
- **Delete** a user
- Simple and readable code with clear comments
- Uses UUIDs for unique user IDs

---

## Requirements

- Node.js (v14 or higher)
- npm (Node Package Manager)

---
